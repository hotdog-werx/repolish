# Installation

## Requirements

- Python 3.11 or later

## Install with pip

```bash
pip install repolish
```

## Install with uv

```bash
uv add repolish
```

## Verify

```bash
repolish --help
```

Both `repolish` (the main CLI, which includes `repolish link` for provider
linking) is included in the package — no additional install steps are required.
